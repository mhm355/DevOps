## Docker Tasks

1. build images maually without using dockerfile or docker-compose
2. build images using dockerfile
3. build images to run in multiple platforms such as intel and arm
4. build images and create containers using docker compose
5. create replicas of the app using docker compose and access them ""1. without nginx  2. with nginx""
6. scan the image, dockerfile, and  docker compose files using "***Dockle***, ***Hadolint***, ***Trivy***, ***Checkov*** , ***Docker Scout*** "
7. create and manage a remote docker host without connecting to the remote machine
8. check the image layers using `dive`
9. push images to public or private registry "dockerhub, nexus, harbor, github"
10. use portainer to manage docker host
11. use netshoot to check the network
12. create containers and use `overlay` , `macvlan` , `ipvlan` networks
